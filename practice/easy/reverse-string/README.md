# Reverse String

## Difficulty: Easy

## Description
### Description
Write a program that takes a string as input and prints the string reversed.

### Input Format
A single line containing the string **S**.

### Output Format
Print the reversed string.

### Constraints
* $1 le \text{Length of S} le 1000$

### Examples
**Example 1:**
Input:
```
hello
```
Output:
```
olleh
```
